## MVVM 클래스 계층 구조
- data : Model, 데이터 작업을 수행하는 곳.
- ui : 사용자에게 데이터를 표시하는데 도움이 되는 Activity, Fragment, ViewModel.
- util : 도우미 클래스 및 함수.
